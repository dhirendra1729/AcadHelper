# 🎓 Acad Helper - Your All-in-One Academic Companion

**Acad Helper** is a modern, high-performance academic tool designed for students who want to streamline their workflow. Built with a sleek **Digital Brutalist** aesthetic, it combines core utilities with cutting-edge AI to help you manage your studies efficiently.

---

## ✨ Key Features

### 📅 1. Smart Time-Table
Never miss a class again. Upload and view your schedule in high resolution with instant access from the home screen.
- **Single-view focus**: No cluttered calendars, just your schedule when you need it.
- **Quick Navigation**: One-tap access to keep you on time.

### 📝 2. AI-Summarizer
Turn hours of reading into minutes of understanding. Simply snap a photo of your notes or textbooks.
- **OCR Integration**: Powered by Google ML Kit to extract text from images with pinpoint accuracy.
- **Gemini AI**: Generates concise, intelligent summaries that capture the essence of your documents.
- **Perfect for**: Lecture notes, complex textbook chapters, and handwritten memos.

### 💻 3. AI-Code Help
Stuck on a bug? Need an algorithm explained? The AI-Code Help section is your personal coding tutor.
- **Multi-Language Support**: Choose from Python, JS, Dart, C++, Java, and more.
- **Quick Queries**: Pre-configured buttons for common coding hurdles.
- **Interactive Chat**: Ask complex questions and get structured code examples in return.

### 📋 4. Assignments (Coming Soon)
A dedicated space to track your tasks, deadlines, and project progress.

---

## 🛠️ Tech Stack
- **Frontend**: Flutter (v3.0+) for high-performance, cross-platform UI.
- **OCR**: Google ML Kit (Vision API).
- **Intelligence**: Google Generative AI (Gemini 1.5 Flash).
- **Aesthetic**: Custom "Dark Navy & Teal" design system.

## 🚀 Getting Started

### 📦 Prerequisites
- **Flutter SDK**
- **Tesseract OCR Engine** (Optional, for Linux development):
  - Linux: `sudo apt-get install tesseract-ocr`

### 📱 Installation & Setup
1. **Clone the Repo**:
   ```bash
   git clone https://github.com/dhirendra1729/AcadHelper.git
   cd AcadHelper/frontend
   ```
2. **Fetch Dependencies**:
   ```bash
   flutter pub get
   ```
3. **Configure API Keys**:
   Open `lib/services/api_service.dart` and add your **Gemini API Key**.
4. **Run the App**:
   ```bash
   flutter run
   ```

---

## 💡 How to Use
1. **Home Screen**: Use the top grid cards to navigate between sections. Check the "Write about yourself" section to track your goals.
2. **Summarizer**: Tap "AI-SUMMARIZER", upload an image, and wait for the AI to process the summary.
3. **Code Help**: Select your language in the sidebar, type your question, and hit send!
